# usage

Usage: ansview.py [-h] [--cat] [--sauce] [file]

positional arguments:
  file                  ANS file (or stdin with -)

options:
  -h, --help            show this help message and exit
  --cat                 Dump raw ANSI to stdout
  --sauce               Show SAUCE metadata including comments
