# usage

_ansview2_0.py [-h] [--sauce] [file]_ <br/>

positional arguments:<br/>
- file                  ANS file (or stdin with -)<br/>

options:<br/>
- -h, --help            show this help message and exit  <br/>
-  --sauce               Show SAUCE metadata including comments<br/>
